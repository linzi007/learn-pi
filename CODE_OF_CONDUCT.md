# 行为准则

Learn Pi 希望成为一个让读者能够安心提问、验证源码和改进课程的学习社区。无论经验、身份、地区、语言、性别、性取向、残障状况、宗教或其他个人特征如何不同，所有参与者都应得到尊重。

## 期待的行为

- 围绕事实、源码、可复现步骤和课程目标讨论问题。
- 对初学者保持耐心，指出问题时解释原因和可行的下一步。
- 尊重不同观点；不同意时批评观点或实现，不攻击个人。
- 发现自己说错时及时更正，并帮助维护准确的课程材料。

## 不接受的行为

- 侮辱、骚扰、歧视、威胁、跟踪或泄露他人隐私。
- 恶意刷屏、推广无关内容、绕过安全限制，或提交含密钥和私人数据的内容。
- 将未经同意的他人作品、会话记录或个人信息带入公开讨论。
- 明知课程为教学简化，仍以人身攻击或重复纠缠方式要求维护者接受无关改动。

## 适用范围

本准则适用于本仓库的 Issue、Discussion、Pull Request、代码评审和其他以本项目名义进行的公开交流。

## 报告与处理

需要保密的行为举报可通过 [GitHub Security Advisory](https://github.com/linzi007/learn-pi/security/advisories/new) 私下提交；这条通道也可用于行为问题，无需在公开页面披露个人信息。请尽量提供发生位置、时间、相关链接和期望的处理方式。

维护者会根据上下文采取提醒、编辑或删除内容、限制互动或禁止参与等措施，并尽可能尊重参与者隐私。公开讨论中的一般分歧应优先在原线程中基于事实解决。
